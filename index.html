Projects

All Deep Learning Image Processing Signal Processing
  • Lung and Heart segmentation from Chest X-ray

    RefineNet is a new semantic segmentation based on ResNet architecture which has achieved state-of-the-art accuracy in PASCAL VOC Challenge. Here I have implemented and fine-tuned the network to segment heart and lung regions in Chest X-rays.

  • Classification of diseases from chest x-ray using deep convolutional neural network

    This is an implementation of popular CNN based architectures i.e. AlexNet, VGG-16, VGG-19, ResNets in disease classification from x-ray images.

  • Lung and Heart segmentation from Chest X-ray using SIFTflow Algorithm

    SIFTflow allows us to warp a model training mask to create a new binary mask of a test image based of SIFT features

  • Sleep Apnea Detection from EEG Signal Analysis

    The objective of my Undergrad Thesis was to detect sleep apnea from EEG Signal Analysis.
